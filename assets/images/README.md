# Character Images

This folder should contain character images for each martial rank in the Pomodojo app.

## Naming Convention

Each rank should have 3 images:
- `{rank_name}_1.png` - Sitting pose (shown when timer is not started)
- `{rank_name}_2.png` - Focusing pose (shown during focus sessions)
- `{rank_name}_3.png` - Break pose (shown during break sessions)

## Rank Images Needed

1. **novice**: `novice_1.png`, `novice_2.png`, `novice_3.png`
2. **apprentice**: `apprentice_1.png`, `apprentice_2.png`, `apprentice_3.png`
3. **disciple**: `disciple_1.png`, `disciple_2.png`, `disciple_3.png`
4. **adept**: `adept_1.png`, `adept_2.png`, `adept_3.png`
5. **master**: `master_1.png`, `master_2.png`, `master_3.png`
6. **grandmaster**: `grandmaster_1.png`, `grandmaster_2.png`, `grandmaster_3.png`

## Usage

These images are displayed on the timer page:
- Position: At the bottom in front of the timer circle
- Size: 120x120 pixels
- Change based on timer state and user's rank

## Placeholder

If an image file is missing, a colored placeholder with the rank icon will be displayed instead.
