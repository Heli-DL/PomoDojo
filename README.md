# Pomodojo App

A Flutter-based Pomodoro timer app with progression tracking, achievements, and focus features.

This app uses Firebase for authentication, Firestore database, and cloud storage.

## Project Structure

- `lib/` - Main application code
  - `features/` - Feature modules (auth, timer, tasks, achievements, etc.)
  - `widgets/` - Reusable widgets
  - `router.dart` - App routing configuration
  - `main.dart` - App entry point
